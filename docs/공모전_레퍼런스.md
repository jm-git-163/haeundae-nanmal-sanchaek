> 국내외 수상작·오픈소스·데이터셋·논문·접근성 표준·참여 메커니즘 총정리.

## 0. 한눈에 — 이 자료로 무엇을 하나

| 축 | 핵심 레퍼런스 | 우리 앱 적용 |
|---|---|---|
| **데이터 근거(가점 최강)** | 부산 공공데이터 축제정보 OpenAPI | 해운대 행사 퀴즈를 **공식 개방데이터**로 채운다 → "실증·신뢰" |

## 2. 부산 공공데이터 (해운대 행사·정보 퀴즈 소스) ★ 실증 근거

- **부산광역시_부산축제정보 서비스 (OpenAPI)** — 축제명·구군·장소·기간·요금·상세·편의시설. 16개 구군 + 시 대표축제 + 민간행사 포함. **해운대 행사 퀴즈 데이터로 직접 사용**.
  https://www.data.go.kr/data/15063500/openapi.do
- **부산광역시_부산관광안내소정보 서비스** — 해운대 등 관광지/안내소 좌표·정보.
  https://www.data.go.kr/data/15063445/openapi.do
- **Big-데이터웨이브(부산시 데이터포털)** — 부산 전용 개방데이터 카탈로그.
  https://data.busan.go.kr/bdip/
- 해운대구 동별 정보는 **해운대구청 공식 홈페이지/문화관광**에서 수집(출처 명기 필수).

> **활용**: 속담 탭 → 해운대 정보 퀴즈로 교체. 데이터 형식 `[질문, 정답, 해설, 등급]`.
> 공식 개방데이터를 쓰면 "데이터 근거 있음 + 지속 갱신 가능"으로 실현가능성/신뢰 점수를 딴다.

## 4. 한국어 AI 모델·데이터 — HuggingFace (음성 읽어주기 / 퀴즈 자동생성)

- **myshell-ai/MeloTTS-Korean** — 경량 오픈소스 한국어 TTS. 어르신용 **문제 읽어주기(음성)** 기능에 적합.
  https://huggingface.co/myshell-ai/MeloTTS-Korean
- **Qwen3-TTS / Fun-CosyVoice 3.0 / Orpheus** — 고품질 다국어(한국어 포함) TTS. **글로벌 다국어 버전** 확장 시.
  https://huggingface.co/Qwen/Qwen3-TTS-12Hz-1.7B-CustomVoice
- **davidkim205/ko-ged** — 한국어 과목별 단답형 문항 데이터. **퀴즈 문항 자동생성/난이도 밸런싱** 참고.
  https://huggingface.co/datasets/davidkim205/ko-ged
- **KorQuAD 1.0** — 한국어 기계독해 표준 데이터. 정보 지문→문항 생성 파이프라인 참고.
  https://arxiv.org/pdf/1909.07005

> **적용**: ①TTS로 저시력 어르신께 문제·해설 음성 지원(접근성 점수) ②LLM+ko-ged로 해운대 정보 지문에서 퀴즈 자동 양산(콘텐츠 확장성).

## 8. 참여 지속 메커니즘 — 글로벌 최상위 사례

- **Duolingo 스트릭(연속) 전략**: 스트릭은 "가장 강력한 리텐션 레버". 7일+ 스트릭 DAU 3,200만. **손실회피** 심리 + **프리즈/복구**(하루 빠져도 안 무너짐) + **위젯 도입 시 몰입 +60%**. DAU/MAU ≈ 37%.
  https://www.lennysnewsletter.com/p/how-duolingo-reignited-user-growth
- **ElliQ 3.0 (iF 디자인상)**: 고령 동반 AI. **외로움 95% 감소**, 하루 30회+ 상호작용, 여러 주정부가 **무료 보급**. → "지자체가 어르신께 무료 보급하는 공익 서비스" 프레이밍의 글로벌 선례.
  https://fuseproject.com/news/elliq-3-0-wins-if-award-redefining-senior-care/
- **BrainHQ (NIH ACTIVE 20년 연구)**: 적응형 속도훈련. 근거기반 설계의 표준(단, 효과는 정직하게).

> **적용**: ①매일 접속 유도용 **연속일 + 관대한 복구** 도입 ②"부산시가 경로당·복지관에 무료 보급하는 어르신 공익 서비스"로 ElliQ식 포지셔닝.
